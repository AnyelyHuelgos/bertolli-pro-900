# Bertolli Pro 900

Este proyecto es una landing page de producto para una cocina a gas profesional 
ficticia de 5 hornillas. Lo desarrollé como prueba técnica frontend donde el 
reto era construir algo visualmente premium usando solo HTML, CSS y JavaScript 
vanilla — sin frameworks, sin librerías, sin atajos.

La página está pensada para una persona que está investigando antes de comprar: 
alguien que quiere ver el producto, entender sus características, resolver sus 
dudas y finalmente cotizar.



## Decisiones de diseño

La referencia visual fueron marcas como Smeg y KitchenAid — cocinas premium 
europeas con una estética limpia, confiable y sin excesos.

**Colores:** Elegí negro carbón y crema porque juntos evocan acero y mármol 
sin verse fríos. El dorado es el único acento — lo usé solo para precios, 
detalles y hover, nunca para texto largo.

**Tipografía:** Playfair Display para los títulos porque tiene personalidad 
sin gritar. DM Sans para el cuerpo porque es legible en cualquier tamaño 
y se ve moderna sin esfuerzo.





## Cómo verlo localmente

1. Clona el repositorio
2. Abre la carpeta en VS Code
3. Click derecho en `index.html` → Open with Live Server
4. Se abre solo en el navegador en `localhost:5500`

No necesita instalación ni dependencias.
